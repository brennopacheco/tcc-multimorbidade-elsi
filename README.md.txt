# Predição de Multimorbidade em Idosos (ELSI-Brasil)

Este repositório contém os códigos utilizados no Trabalho de Conclusão de Curso (TCC) em Engenharia da Computação, com foco na predição de multimorbidade em idosos por meio de técnicas de classificação multirrótulo.

## Estrutura do Projeto

- `Tratamento_Dados.R` — Limpeza, recodificação e preparação dos dados
- `porcentagem_doencas.R` — Análise descritiva e prevalência das doenças
- `identificadores.R` — Mapeamento e identificação das variáveis
- `Geracao_Modelagem.R` — Modelagem preditiva e avaliação
- `resultados/` — Tabelas, figuras e saídas dos modelos

## Base de Dados
Estudo Longitudinal da Saúde dos Idosos Brasileiros (ELSI-Brasil) — 2ª onda.

⚠️ Os dados originais não são disponibilizados neste repositório por questões éticas.
